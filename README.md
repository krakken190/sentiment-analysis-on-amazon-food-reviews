# sentiment-analysis-on-amazon-food-reviews
This project involves analyzing sentiment in Amazon Fine Food reviews using natural language processing techniques. The dataset used is a collection of over 500,000 food reviews from Amazon, spanning a period of more than 15 years.
This repository contains the code and data for a project on sentiment analysis of Amazon Fine Food reviews. The project involved analyzing millions of customer reviews to gain insights into how customers feel about the food products they purchase on the Amazon platform.

The repository includes Python code for conducting the sentiment analysis, as well as Jupyter notebooks for visualizing the results. The dataset used for the analysis is also included in the repository, along with a README file that explains how to use the code and data.

This project is intended to be a resource for anyone interested in sentiment analysis or customer feedback analysis, as well as businesses looking to improve their customer experience on Amazon. The code can be used as a starting point for conducting similar analyses on other datasets or platforms.

Feel free to use the code and data for your own projects, and please let us know if you have any feedback or suggestions for improvement.
